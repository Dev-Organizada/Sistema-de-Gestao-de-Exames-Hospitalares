# Sistema de Gestão de Exames Hospitalares
---
Este projeto foi desenvolvido com o objetivo de **simular o ciclo completo de um exame médico** — desde a solicitação feita pelo médico até a liberação do resultado para o paciente.

A finalidade é **demonstrar habilidades em modelagem de dados, SQL e PL/SQL**, aplicadas a um cenário real de gestão hospitalar, incluindo controle de autorizações, agendamentos, resultados e auditoria de processos.

O principal objetivo é **otimizar o gerenciamento das informações de exames**, garantindo integridade, rastreabilidade e eficiência no fluxo de dados, além de servir como portfólio técnico para demonstrar meus conhecimentos em banco de dados Oracle.
